# Password Manager

Este é um gestor de passwords desenvolvido em Python, focado em facilidade de uso e armazenamento estruturado de credenciais.

## Tecnologias
- Python 3.x
- Tkinter (Interface Gráfica)
- Pyperclip (Gestão de clipboard)

## Funcionalidades
- Geração de Passwords: Cria passwords complexas automaticamente.
- Armazenamento: Grava as credenciais num ficheiro local.
- Interface: Design simples e funcional usando Tkinter.

## Aviso de Segurança
Este é um projeto educativo. Não utilize este programa para guardar passwords reais. Os dados são armazenados localmente e, nesta fase, não possuem encriptação.

## Como correr
1. Certifique-se de que tem o Python instalado.
2. Instale a biblioteca pyperclip: `pip install pyperclip`
3. Execute o comando: `python main.py`
